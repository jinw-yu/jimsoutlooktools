# 更新日志

本文件记录项目的所有版本更新历史。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本 2.0.0](https://semver.org/lang/zh-CN/)。

---

## [v1.0.2] - 2025-03-02

### 修复
- 优化内存管理，解决大日期范围（如 2/1-3/2）处理时内存不足问题
  - 使用 for 循环代替 foreach，手动释放 COM 对象
  - 每处理 50 封邮件强制垃圾回收
  - 每 100 封邮件刷新 UI，避免假死

## [v1.0.1] - 2025-03-02

### 优化
- 在 GUI 各处添加 `jimsoutlooktools` 品牌标识和版本号显示
  - Outlook 工具栏名称改为 "jimsoutlooktools"
  - 对话框标题显示版本号，如 "jimsoutlooktools v1.0.1 - 保存邮件附件"
  - 对话框顶部添加蓝色品牌标题
  - 进度条窗口标题显示版本号
  - 所有提示框标题统一显示品牌和版本号
  - 按钮悬停提示显示版本信息
- 优化版本号管理，统一使用常量定义

---

## [v1.0.0] - 2025-03-01

### 新增
- 初始版本发布
- 批量保存 Outlook 邮件附件功能
  - 支持选择日期范围（起始日期、结束日期）
  - 自动按 `年月(yyyyMM)` 创建子文件夹分类
  - 文件命名格式：`时间戳(精确到毫秒)_原文件名`
  - 示例：`20250302_143052_123_invoice.pdf`
- 智能过滤功能
  - 自动跳过小于 100KB 的图片文件（邮件内联图标、表情等）
  - 只保存真正的附件（文档、照片、压缩包等）
- 防重复机制
  - 自动检测已存在的文件并跳过
  - 保存完成后显示统计：已保存数、跳过数
- 用户界面
  - Outlook 工具栏集成
  - 统一的保存对话框（路径选择 + 日期范围）
  - 实时进度条显示
  - 清晰的标签提示（起始日期、结束日期）

### 技术栈
- C# / .NET Framework 4.7.2
- VSTO (Visual Studio Tools for Office)
- Microsoft Outlook 插件开发
